Connects to Sensibo cloud and sends updates for the selected pod

Configuration

An API needs to be obtained from Sensibo

Properties

pod selector
The pod to get the measurements from. Click on magnifying glass to get list of available pods from sensibo. This may take a moment to resolve from the server

Inputs

payload string | buffer
Payload is not currently assigned to a function
msg.onstring/binary
Turn the Airconditioning unit on and off. true/false. Bug fixed so both types will accepted
msg.swingstring
Sets the swing mode.
msg.modestring
Set the operating mode: cool/heat/fan/dry/auto'
msg.fanlevelstring
Set the fan level: quiet/low/medium low/medium/medium high/high/auto. Note: medium high/low may not be supported on all units
msg.targetTemperaturenumber
Sets the target temperature to reach.'

Outputs

  1. Standard output
    payload object
    Status of the request to Sensibo
  2. Standard error
    payload string
    Will return the error received from the Sensibo server
